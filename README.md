# Video Steganography
A system that can hide or show secret messages through video media.

## Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirement libraries.

```bash
pip install -r requirements.txt
```

## Usage
```bash
python stego_gui.py
```
